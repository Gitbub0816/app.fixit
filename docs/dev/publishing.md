# Publishing

## Web (Blazor)
- Build and publish `src/FixItQC.Web` to app.fixitqc.com hosting target.

## API
- Build and publish `src/FixItQC.Api` with environment settings.

## MAUI Hybrid
- Build `src/FixItQC.Mobile` for Windows, iOS, Android.
- Apply platform signing/certificates during release.
